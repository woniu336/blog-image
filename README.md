# Github图床使用



配合图床工具: [PicList](https://github.com/Kuingsmile/PicList)


`PicList` 是一款基于`PicGo` 深度二次开发,云储存/图床管理和文件上传客户端工具


完美兼容PicGo


官网: https://piclist.cn
